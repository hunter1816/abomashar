⚠️ تنبيه: طريقة التشغيل غير صحيحة

يبدو أنك تحاول تشغيل البرنامج بالنقر المباشر على الملف (File Protocol).
هذا التطبيق يستخدم تقنيات حديثة تتطلب خادماً محلياً (Local Server) لتعمل بشكل صحيح ولتتمكن من الاتصال بالذكاء الاصطناعي.

الحل المقترح (لأجهزة الكمبيوتر):
  • استخدم برنامج VS Code مع إضافة "Live Server".
  • أو استخدم Python في مجلد المشروع: python -m http.server
  • أو ارفع الملفات على استضافة ويب تدعم تطبيقات React.
